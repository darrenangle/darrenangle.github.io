    Submittr

    Submittr is an open source submissions manager for literary magazines written in Backbone.

    It utilzes Parse.com's JavaScriptSDK to offer a free alternative to young literary magazines who shouldn't have to choose between shelling out $$$ for a mediocre submissions manager or sifting through thousands of emails by hand.

    Submittr uses efficient queries and file-size restrictions to keep your monthly API requests below the threshold that triggers costs. When Submittr is complete, magazines will be able to handle 5,000+ submissions a month for free.
